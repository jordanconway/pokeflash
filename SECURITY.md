# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately. 

Do not open a public GitHub issue for security vulnerabilities. Instead, please report vulnerabilities by contacting the maintainer directly at:

**Email:** [jordan@jordanconway.com](mailto:jordan@jordanconway.com)

Please include as much detail as possible, including:
- A description of the vulnerability and its potential impact.
- Steps to reproduce the issue (including any proof-of-concept scripts or screenshots).
- Any details regarding your environment or configuration.

We will acknowledge receipt of your report within 48 hours and work to resolve the issue as quickly as possible.
